Model Architecture
MFCM-DTI · SG_GNN · LTDTI · MLP Decoder
SG_GNN (Drug)
Drug Module
Uses SAGEConv and GATConv from Graph Neural Networks to capture complex hidden features of drug molecular structures. Neighbor sampling aggregates multi-hop structural topology, physicochemical properties, and electronic characteristics.
📐 SAGEConv — Inductive neighbor aggregation
🎯 GATConv — Attention-weighted edges
🔀 Feature Fusion — Concatenation output
LTDTI (Protein)
Protein Module
Integrates a Linear Encoder, Self-Attention Mechanism, and Linear Decoder to improve representation of protein sequences. Maps high-dimensional protein features to compact, interaction-aware embeddings.
📥 Linear Encoder — Dimension mapping
👁 Self-Attention — Sequence context
📤 Linear Decoder — Feature decoding
MLP Decoder
Classifier
After feature concatenation from both drug and protein pathways, a Multi-Layer Perceptron with ReLU activations and sigmoid output predicts the final drug-target interaction probability score.
🔗 Concatenation — Feature fusion layer
🔢 Hidden Layers — Non-linear learning
📊 Sigmoid — DTA probability output
Overall 3-Stage Framework
End-to-End
Stage 1 · Protein
Transformer network trains on Anc2Vec + manual features from STRING PPI database. Ablation confirms fusion outperforms single-feature approaches (93.91% vs 92.88%).
Stage 2 · Drug
GCN refines drug embeddings from Mol2Vec and deep neural networks. DDI dataset confirms feature embedding + Mol2Vec fusion achieves 91.37% accuracy vs 87.14% single-feature.
Stage 3 · Prediction
MFCM-DTI fuses protein + drug features via concatenation and MLP. Validated on BioSNAP, BindingDB, Human datasets with consistent state-of-the-art performance.
Feature Extraction
Multimodal representations for proteins and drugs
Protein Feature Extraction
Dual-Mode
Anc2Vec Embeddings
Maps protein sequences to a low-dimensional vector space preserving ancestor relationships in Gene Ontology (GO). Enables identification of similarity and functional relationships among protein sequences.
Handcrafted Sequence Features
Manually designed features include amino acid composition and secondary structure, reflecting the physical and chemical properties of proteins. Used with PseAAC and APsaaC encoding modules.
Transformer Refinement
Multi-head attention Transformer network fuses both feature types, trained on human PPI data from the STRING database. Accuracy: 93.91% (fused) vs 89.18% (handcrafted alone).
Drug Feature Extraction
Dual-Mode
Mol2Vec Embeddings
Unsupervised word2vec-inspired approach converting drug molecules into low-dimensional vectors. Effectively captures topological structure and chemical properties of molecular substructures.
Deep Neural Network
Learns abstract and complex representations of drug features including structural topology, electronic properties, and physicochemical characteristics beyond what Mol2Vec captures.
GCN Optimization
Graph Convolutional Network fuses DNN + Mol2Vec embeddings trained on DDI dataset. Combined features achieve 91.37% accuracy and 97.13% AUC — best among all ablation variants.
